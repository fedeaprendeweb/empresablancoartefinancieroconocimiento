Arquitectura Cliente-Servidor

10 min

Las aplicaciones web siguen el modelo cliente-servidor. El cliente (navegador) solicita recursos al servidor, que procesa la solicitud y devuelve una respuesta.

Frontend: Todo lo que se ejecuta en el navegador (HTML, CSS, JavaScript).

Backend: La lógica del servidor, bases de datos y APIs.

Full Stack: Desarrollo tanto de frontend como backend.

HTML en Profundidad

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el significado del contenido web mediante el uso de elementos y etiquetas.

Estructura Básica de un Documento HTML

15 min

Todo documento HTML sigue una estructura básica que incluye la declaración DOCTYPE, elementos html, head y body.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
    <!-- Enlaces a CSS -->
</head>
<body>
    <header>
        <h1>Encabezado principal</h1>
    </header>
    <main>
        <p>Párrafo de texto</p>
    </main>
    <footer>
        <p>Pie de página</p>
    </footer>
    <!-- Scripts JavaScript -->
</body>
</html>

Elementos Semánticos HTML5

12 min

HTML5 introdujo elementos semánticos que describen mejor el contenido y mejoran la accesibilidad y SEO:

  • <header> - Encabezado de sección o página
  • <nav> - Navegación principal
  • <main> - Contenido principal único
  • <section> - Sección temática de contenido
  • <article> - Contenido independiente
  • <aside> - Contenido relacionado indirectamente
  • <footer> - Pie de sección o página

CSS Avanzado

CSS (Cascading Style Sheets) se utiliza para describir la presentación de documentos HTML, incluyendo colores, diseños y fuentes. Permite separar el contenido de la presentación.

Fundamentos de CSS

18 min

CSS funciona mediante reglas que consisten en selectores y declaraciones. Las declaraciones contienen propiedades y valores.

/* Selector de elemento */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 0;
}

/* Selector de clase */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

/* Selector de ID */
#header {
    background-color: navy;
    color: white;
    text-align: center;
    padding: 20px;
}

/* Selector de atributo */
input[type="text"] {
    border: 1px solid #ccc;
    padding: 8px;
}

/* Pseudo-clases */
a:hover {
    color: red;
    text-decoration: underline;
}

/* Media queries para responsive design */
@media (max-width: 768px) {
    .container {
        padding: 10px;
    }
}

Flexbox y Grid Layout

20 min

CSS Flexbox y Grid son sistemas de diseño modernos que facilitan la creación de layouts complejos y responsive.

Flexbox (Diseño unidimensional):

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.item {
    flex: 1 0 200px;
    margin: 10px;
}

CSS Grid (Diseño bidimensional):

.container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

.item {
    grid-column: span 1;
}

JavaScript Completo

JavaScript es un lenguaje de programación que permite crear contenido dinámico e interactivo en las páginas web. Se ejecuta en el navegador del usuario.

Fundamentos de JavaScript

25 min

JavaScript incluye conceptos como variables, funciones, objetos, arrays y control de flujo.

// Variables (let, const)
let nombre = "Juan";
const edad = 30;

// Funciones
function saludar(nombre) {
    return `Hola, ${nombre}!`;
}

// Arrow functions (ES6)
const sumar = (a, b) => a + b;

// Arrays y métodos
const numeros = [1, 2, 3, 4, 5];
const duplicados = numeros.map(num => num * 2);

// Objetos
const persona = {
    nombre: "Ana",
    edad: 25,
    saludar: function() {
        return `Hola, soy ${this.nombre}`;
    }
};

// Control de flujo
if (edad >= 18) {
    console.log("Mayor de edad");
} else {
    console.log("Menor de edad");
}

// Bucles
for (let i = 0; i < numeros.length; i++) {
    console.log(numeros[i]);
}

Manipulación del DOM

15 min

El DOM (Document Object Model) permite a JavaScript interactuar con los elementos HTML de una página.

// Seleccionar elementos
const elemento = document.getElementById("miId");
const elementos = document.querySelectorAll(".miClase");

// Cambiar contenido
elemento.textContent = "Nuevo texto";
elemento.innerHTML = "<strong>Texto en negrita</strong>";

// Cambiar estilos
elemento.style.color = "red";
elemento.classList.add("nueva-clase");

// Crear elementos
const nuevoElemento = document.createElement("div");
nuevoElemento.textContent = "Soy nuevo";
document.body.appendChild(nuevoElemento);

// Event listeners
elemento.addEventListener("click", function() {
    alert("Elemento clickeado!");
});

// Formularios
const formulario = document.getElementById("miFormulario");
formulario.addEventListener("submit", function(event) {
    event.preventDefault();
    const datos = new FormData(formulario);
    // Procesar datos...
});
